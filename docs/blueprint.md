# **App Name**: Jijau SchoolConnect

## Core Features:

- Secure User Authentication: Implement robust login for Admin, Principal, and Teachers, securing access with JWTs.
- Student Profile Management: CRUD operations for student profiles including academic details, contact information, and automated caste category conversion based on defined logic.
- Teacher Profile Management: CRUD operations for teacher profiles, including subject taught, experience, contact info, photo upload, and automated caste category conversion.
- Central Administration Panel: A unified interface for Admin/Principal to manage all student and teacher records, create teacher login accounts, and oversee school data.
- Teacher Payroll & Payment Tracker: System to record teacher payments, including amount, status (paid/pending), mode (cash/online), payment date, and mandatory online payment screenshot uploads.
- Personalized Dashboards: Role-specific dashboards providing Admins/Principals with high-level statistics and Teachers with their profile, payment history, and assigned student lists.
- AI Communication Tool: An AI tool to generate personalized welcome messages for new students or teachers, or create informational blurbs for school announcements based on input data.

## Style Guidelines:

- Primary color: A deep, educational blue (#1F58B3) evoking wisdom and reliability.
- Background color: A very light, subtle blue-gray (#EEF3F5) providing a clean and professional base.
- Accent color: A vibrant aqua-cyan (#16B9DE) for interactive elements and key highlights to draw attention.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and objective feel across all text sizes.
- Utilize a consistent set of clean, modern outline-style icons to enhance clarity and user experience, especially for administrative tasks and profile management.
- Design a responsive layout that adapts seamlessly to both mobile and desktop screens, featuring a simple, intuitive navigation structure and a top-right user profile display with photo and name.
- Incorporate subtle and smooth transition animations between pages and sections, as well as clear feedback animations for user interactions like button clicks or form submissions.
